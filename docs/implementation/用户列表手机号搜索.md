# 用户列表手机号搜索开发说明

关联 Issue: https://github.com/yuzhe/aiTest/issues/2
关联 PRD: docs/requirements/2026-07-用户列表手机号搜索.md

## 实现内容

实现用户列表按手机号精确搜索能力。

## 处理规则

- 输入为空时展示全部用户列表。
- 输入前后空格时，先去除空格再查询。
- 输入非 11 位手机号时，不发起查询，并提示“请输入 11 位手机号”。
- 输入完整手机号时，按手机号精确查询。
- 手机号无匹配结果时展示空列表。
- 接口异常时展示“查询失败，请稍后重试”。

## 验收结果

- [ ] 输入完整手机号后，可以筛选出对应用户。
- [ ] 输入不存在的手机号时，页面展示空列表。
- [ ] 输入前后带空格的手机号时，系统自动去除空格后查询。
- [ ] 输入非 11 位手机号时，不发起查询，并提示“请输入 11 位手机号”。
- [ ] 清空手机号搜索条件后，恢复展示全部用户列表。
